# ⚡ elcView — עוזר ביקורת לחשמלאים

אפליקציית ווב (HTML + JavaScript, ללא build) לחשמלאים: מעלים PDF של תכנית חשמל,
המערכת מזהה את השקעים והחדרים, ומפיקה טבלת ביקורת עם ייצוא לאקסל.

## יכולות (שלב א')

- **העלאת שרטוט PDF** — רינדור עם pdf.js, זום, גרירה ותמיכה במגע.
- **זיהוי שקעים אוטומטי** — תוויות הגובה (`H=60`, `H=120`...) וסימוני `TV`/`T`
  נקראים משכבת הטקסט של ה-PDF (ייצוא אוטוקאד) עם מיקום מדויק.
- **זיהוי חדרים ב-OCR** — Tesseract.js בעברית (מצב טקסט פזור) מציע שמות
  חדרים; המשתמש מותח את מלבני החדרים לגבולות האמיתיים.
- **עריכה על גבי השרטוט** — הוספה/גרירה/מחיקה של שקעים וחדרים, גובה מהרצפה,
  מרחק מפינת הקיר, מעגל וסוג נקודה.
- **טבלת ביקורת** — מקובצת לפי חדר; עריכה מותרת רק בעמודות "בוצע" ו"הערות"
  (נשמר אוטומטית); ייצוא לקובץ אקסל RTL.
- **בסיס נתונים** — Supabase (פרויקטים, חדרים, שקעים + Storage לשרטוטים).
- **מסך ביקורת** — שלד לשלב ב': בחירת חדר ופתיחת מצלמה אחורית.

## הרצה

```bash
python -m http.server 8123
# ולפתוח http://localhost:8123
```

## מבנה הקוד

```
index.html            מסכים: פרויקטים / שרטוט / טבלה / ביקורת
css/style.css         עיצוב RTL, מותאם למובייל
js/
  app.js              בקר ראשי (class App)
  config.js           הגדרות: Supabase, מילון חדרים, ברירות מחדל
  models.js           Project / Room / Outlet
  services/
    pdfService.js     טעינה ורינדור PDF, חילוץ טקסט (pdf.js)
    detector.js       מנוע הזיהוי: H=xx ⇒ שקעים, שיוך לחדרים
    ocrService.js     זיהוי שמות חדרים (Tesseract.js heb, PSM 11)
    db.js             Supabase: CRUD + Storage
    excelExporter.js  ייצוא xlsx (SheetJS)
  ui/
    planViewer.js     קנבס + שכבת עריכה אינטראקטיבית (זום/פאן/גרירה)
    tableView.js      טבלת הסיכום
    inspectionView.js מסך ביקורת עם מצלמה (שלד לשלב ב')
```

## מפת דרכים

- **שלב ב'** — זיהוי קופסה כתומה במצלמת הטלפון ומדידת גובה/מרחק מפינה
  (קנה מידה לפי קופסת 55 מ"מ סטנדרטית).
- **שלב ג'** — השוואת המדידות לשרטוט.
- **שלב ד'** — סריקת וידאו רציפה עם סימון ✓/✗ אוטומטי לכל נקודה.
